About ADAOZ

ADAOZ has been minting blocks since 20 February 2021. A small, well-maintained Australian stake pool focused on reliability, security, and contributing back to Cardano rather than chasing scale at any cost.

ūüĆĪ

Powered by renewables

Infrastructure runs on renewable-energy data centres. Cardano's proof-of-stake already cuts the footprint to a fraction of proof-of-work; we go further by choosing greener hosts.

ūüĒź

Hardened security

Cold keys stored offline, hardware-backed signing, restricted SSH access, continuous monitoring on every relay. The block-producer node is not directly internet-exposed.

Redundant relays

Multiple geographically distributed relays with health monitoring. If one drops, your delegation keeps minting on the rest.

The operator

Peter Bui

Cardano Ambassador ¬∑ Midnight Ambassador ¬∑ Founder, Mesh With Us ¬∑ Host, Learn Cardano

I started ADAOZ in February 2021 because I wanted Australians to have a local pool to delegate to, and because I wanted to learn how Cardano really worked from the inside out. Five years later I'm still running the pool, still publishing on Learn Cardano, and building Cardano and Midnight apps via Mesh With Us.

I'm also a Midnight Ambassador, covering everything happening on Cardano's new privacy-focused sidechain as it rolls out.

Delegating to ADAOZ earns you ADA rewards but also helps fund the time it takes to review every governance action, every new tool, interview projects building on Cardano, and turn that into content other people can learn from.
